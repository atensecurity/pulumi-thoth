# Changelog

All notable changes to `pulumi-thoth` are documented in this file.

## 0.1.0 - 2026-05-02

- Introduced official `thoth` Pulumi provider bridged from `terraform-provider-thoth`.
- Added provider binaries (`pulumi-resource-thoth`) and tfgen entrypoint (`pulumi-tfgen-thoth`).
- Added Node.js and Python provider-native examples.
- Added monorepo publish workflow for signed tag release automation.
- Added public release workflow and GoReleaser packaging for Pulumi plugin artifacts.
